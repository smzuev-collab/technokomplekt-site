# GEOSIDEAL

Готовый демонстрационный сайт на Next.js.

## Запуск на Windows

Откройте терминал в папке проекта и выполните:

```powershell
npm.cmd install
npm.cmd run dev
```

Сайт откроется по адресу:

```text
http://localhost:3000
```

## Проверка production-сборки

```powershell
npm.cmd run build
npm.cmd run start
```

## Что заменить перед публикацией

Контактные данные находятся в файле:

```text
lib/site.ts
```

Нужно заменить:

- телефон
- электронную почту
- адрес шоурума
- время работы

Форма на странице контактов сейчас открывает почтовую программу пользователя. Для автоматической отправки заявок нужно подключить CRM, API или сервис обработки форм.

Текст политики конфиденциальности в `app/privacy/page.tsx` является шаблоном и требует юридической адаптации.
